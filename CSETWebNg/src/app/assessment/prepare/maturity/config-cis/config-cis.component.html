Cyber Infrastructure Survey Configuration

Select Baseline Survey

Optionally select another CIS assessment to serve as a "baseline" for comparison. The baseline section scores will be displayed alongside the current assessment's scores during questioning. In the reports, baseline answers will be included for comparison.

To remove a baseline survey, select the top option, "No Baseline Assessment."

Import CIS Answers

Import all answers from another CIS survey. All survey answers, comments and artifacts currently in the assessment will be overlaid.
