Merge

 
Once the assessment file table is complete, click the Merge tab to begin the Merge function. Depending on the number and size of the selected assessment files, analysis may take a few moments.
 
The merge function often requires additional input from the user before the aggregation can be completed. Because multiple assessments will be combined into a single assessment, each question may only have one answer. If a question was answered differently among the assessments to be merged, the correct or final answer must be identified. This is done under the Merge tab.
 
 
Merge Assessments
 
Click the Merge tab to resolve the problem of different answers being given for the same question. Click a subject area or category to open it. The table shows the problem questions and allows the user to select the desired answer. If the question was answered in the default assessment, that answer will be prepopulated in the table. To keep that answer, do nothing. To change that answer, click another radio button.
 
If the question was not answered (left blank) in the default assessment, the answer provided by the second assessment in the Load Multiple screen list will be shown. If not answered in the second assessment, the answer from the third assessment will be used, and so on.
 
If no answer is prepopulated, that means it was not answered in any of the assessments and must be answered here. If not, it will count as a failed question during assessment analysis and in the printed reports. Continue the process until all the problem questions are resolved. Figure 1 shows an example screen with Access Control opened for resolution.
 
Figure 1. Merge Assessments
 
To save time, the option to accept all answers shown under a subject area or category is provided. Click the check box labeled Resolve to accept the prepopulated answers to the questions under that topic.
 
The checkboxes to the left of the questions are to help the user keep track of which questions have been resolved and which have not. A checkmark is automatically added if an answer if changed. A checkmark can be added manually if the prepopulated answer is acceptable.
 
If the resolution step is skipped altogether and no resolutions are made, the Merge function will use the answers shown on this screen. That means it will accept the answer provided by the default assessment if there was one, or leave the question blank if it was not answered in the default assessment. Blank or skipped answers are considered failed or noncompliant during analysis and in the printed reports.
 
 
Merge Question Detail Information
 
Clicking the paper icon in the column next to the question text will open the Merge Question Detail Information screen. Here you can see and edit the comments or alternate text that were entered in any of the subject assessments for that question. You can also see any documents that were saved with a particular assessment and whether the question was marked for review in any of the assessments. The reconciled comments and alternate text entered here will be included in the final merged assessment. Figure 2 shows an example screen.
 
Figure 2. Merge Question Detail Information
 
 
Question Indicators
 
A unique set of indicators are available for display above the paper icon in the column next to the question. The indicators are represented by small colored boxes placed in different locations above the icon. These are the same indicators that may be seen on the Questions screen for a normal assessment. Figure 3 provides an example.
 
   
Figure 3. Question Indicators.
 
The indicators represent:
 
Top (Red) - This question has been marked for review.
 
Center (Yellow) - This question has comments associated with it.
 
Bottom (Blue) - This question has one or more documents associated with it.
 
 
Create Assessment
 
Because the merge function creates a new assessment file rather than a series of charts, there is no button to generate a report. Instead, create assessment buttons allow you to name and save the merged information as a new assessment that can be opened in CSET just like any other assessment.
 
Create Merged Assessment: Clicking Create Merged Assessment allows you to name and save the file.
 
Create and Open Merged Assessment: Clicking Create and Open Merged Assessment allows you to name and save the file which then automatically opens in CSET.
 
 
Changing Assessment Files
 
Changing assessment files while in the Merge process will negate any resolution of question differences that have been completed. A warning message will be displayed if assessment files are added or removed once the Merge process has begun.
