CMMC

 

CMMC has the following unique features:

 
CMMC Target Selection: This topic is covered in the Target Level Selection section.
 
 
 
Maturity Questions:
 
 
 
 
Results: This section shows the individual assessment results for the following data:
 
  • Target and Achieved Levels: This graph shows the results for the current CMMC assessment target level vs. the CMMC level reached post-assessment.
 
 
 
  • Level Drill Down:
     
CMMC Bar Chart: Details the percentage of compliance for each level assessed.
 
  • CMMC Pie Chart: Displays compliant vs. non-compliant percentages for each level assessed.
 
 
 
 
 
  • Compliance Score: This graph shows the total compliance score for the set target level broken down by total practices satisfied per level.
 
 
 
 
  • Detailed Gaps List:
     
  • Gaps Graph: This section shows the gap analysis for each domain.
 
 
 
 
  • Gaps List Scorecard: This section shows each CMMC domain with the individual practices detailed in list format after each domain-specific sub heading. The compliance score for each individual practice is shown on the right, along with the associated cumulative score for each practice and the percent complete for each CMMC domain.
 
 
 
 
 
Report: This section includes the Executive Summary and Site Summary specific to CMMC.
